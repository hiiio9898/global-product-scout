# Spec 30：1688数据来源标注

**版本**：v1.0
**状态**：📋 待实现
**创建日期**：2026-06-10
**前置依赖**：Spec 4（1688比价）

---

## 1. 需求描述

### 1.1 背景
1688比价有三级降级：真实抓取 → AI估算 → 本地规则。但用户看到的价格没有任何来源标记，可能把AI估算误认为真实采购价。

### 1.2 核心需求
1. 1688查询结果明确标注数据来源
2. 三级来源用不同颜色/图标区分
3. 用户可判断数据可信度

---

## 2. 系统设计

### 2.1 改动文件清单

| 文件 | 改动类型 | 说明 |
|------|---------|------|
| `src/scraper_1688.py` | **修改** | 返回值增加 `source` 字段（"live"/"ai_estimate"/"local_rule"） |
| `app.py` | **修改** | 1688结果卡片展示来源标记 |

### 2.2 来源标记UI

```python
source = result_1688.get("source", "unknown")
SOURCE_LABELS = {
    "live":        ("🟢 实时抓取", "数据来自1688实时页面"),
    "ai_estimate": ("🟡 AI估算",  "价格由AI根据产品信息推估"),
    "local_rule":  ("🟠 本地规则", "价格基于品类经验倍率计算"),
}
label, tooltip = SOURCE_LABELS.get(source, ("⚪ 未知", ""))
st.caption(f"{label} — {tooltip}")
```
